# Scoreboard-System
Controllable scoreboard for OBS Overlay (Soccer, Pool, Hockey etc)

# Install
- `download the file`
- `npm i socket.io electron`
- `npm start`

# Add to OBS
- Add new browser
- Set url to `http://localhost:3000/` for scoreboard
- Set url to `http://localhost:3000/control.html` for scoreboard controls
